## Product Overview

实现信息图表前端网页的加载页面功能，提供流畅的用户上传体验和视觉反馈

## Core Features

- 点击确认上传按钮后显示加载页面
- 灰色半透明蒙层覆盖整个网页
- 浮动的番茄emoji动画效果
- loading...文字提示
- 5秒后自动退出加载状态
- 上传前显示"等待您的投喂"提示文字

## Tech Stack

- 前端框架: React (基于现有项目)
- 状态管理: React Hooks (useState, useEffect)
- 样式: CSS动画和过渡效果
- 定时器控制: setTimeout
- 事件处理: onClick事件监听

## Design Style

采用简洁现代的加载页面设计，使用半透明灰色蒙层确保视觉层次清晰，配合浮动动画营造活泼的加载体验。
